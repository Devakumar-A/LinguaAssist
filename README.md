# 🤟 Nagish India – Accessible Communication Platform

An accessibility-focused mobile application designed to help hearing and speech-impaired individuals communicate effectively through real-time speech transcription, text-to-speech conversion, and multilingual communication support.

Developed as a solution for Smart India Hackathon 2024.

## 📖 Project Overview

Communication barriers often make daily interactions difficult for individuals with hearing and speech impairments.

This project aims to provide an accessible communication platform that enables users to participate in conversations using speech-to-text and text-to-speech technologies.

The application supports multilingual communication and focuses on creating an inclusive environment where users can communicate independently and effectively.

---

## 🎯 Problem Statement

People with hearing and speech disabilities face difficulties while using traditional communication applications.

Challenges include:

- Difficulty understanding spoken conversations
- Difficulty expressing responses verbally
- Language barriers
- Dependence on interpreters or assistants
- Limited accessibility features in existing communication tools

This project addresses these challenges by providing real-time communication assistance.

## 🚀 Features

### 🎙 Real-Time Speech-to-Text

Converts spoken words into readable text in real time.

Benefits:

- Helps hearing-impaired users understand conversations
- Displays captions instantly during calls and interactions

---

### 🔊 Text-to-Speech Conversion

Converts typed text into spoken audio.

Benefits:

- Helps speech-impaired users communicate effectively
- Enables smooth two-way conversations

---

### 🌐 Multilingual Support

Supports communication in multiple Indian languages.

Benefits:

- Improves accessibility for diverse users
- Removes language barriers

## 🔄 Application Flow

1. User speaks into the application.
2. Audio is captured.
3. Speech-to-Text service converts speech into text.
4. Text is displayed on screen.
5. User can respond through typing.
6. Text-to-Speech service converts text into voice.
7. Communication continues seamlessly.
